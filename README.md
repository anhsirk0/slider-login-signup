# Installation

```bash
git clone https://github.com/anhsirk0/slider-login-signup.git
```

```bash
cd slider-login-signup
```

# Running the app

## First to download all dependencies run (in the project root , where package.json is located)
```bash
npm install
```

## Finally start the server via
```bash
npm start
```
